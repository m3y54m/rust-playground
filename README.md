# Rust Playground

## Build and Run

### Using `rustc` (Not recommended):

```
rustc src/main.rs
./main 
```

### Using `cargo`:

Build without running:

```
cargo build
```

Build and run:

```
cargo run
```

Only check the source code:

```
cargo check
```
